# 📚 Conteúdo POO

### Este repositório contém exercícios e exemplos práticos desenvolvidos durante o estudo de Programação Orientada a Objetos (POO) utilizando Java.

### O objetivo é reforçar conceitos fundamentais da POO por meio de implementações simples e didáticas.

## 🚀 Conteúdos abordados

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Métodos
- Construtores
- Métodos estáticos
- Recursividade

## 🛠️ Tecnologias utilizadas

Java
IDE (Eclipse / IntelliJ / NetBeans)

## 📂 Estrutura do projeto

### Os exercícios estão organizados em pastas, cada uma abordando um conceito específico da POO.

#### Exemplos:

- Exercícios de Herança
- Exercícios de Métodos
- Exercícios de Encapsulamento
- Exercícios com Classes

## ▶️ Como executar

### Clone o repositório:

git clone https://github.com/vLamass/conteudo_poo.git\
Abra o projeto em sua IDE Java preferida\
Compile e execute os arquivos .java
